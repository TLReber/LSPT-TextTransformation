Installation
============
- Using python 3.7 though 3.6 should work

For linux:

.. code-block:: bash

    python -m venv .tt-env
    source .tt-env/bin/activate
    pip install -e

Running A Node
==============

After installing:

.. code-block:: bash

    text-trans-node
